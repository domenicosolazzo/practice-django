INSTALLATION
==============
- Install Pip
- Install virtualenv
- Create a new virtualenv (e.g. virtualenv venv --distribute)
- Activate the virtualenv (e.g. source venv/bin/activate)
- Install Django using pip (e.g. pip install Django)
